Steps to Setup and Start a React UI Application

Install Node.js
React applications require Node.js to manage dependencies and run development tools.
Download & Install
• Go to: https://nodejs.org
• https://nodejs.org/en/download
• Download the LTS (Long Term Support) version
• Complete the installation using default settings
Verify Node.js Installation
After installation, open Terminal / Command Prompt and run:
node -v


Navigate to the React Project Folder & install dependencies
Install all required dependencies de�ined in package.json: npm install
PS D:\AI\claude-code-udemy-eazy-bytes\Section-3\Chapter-27\start\job-portal-ui> npm install


Start the Development Server
Run the React application in development mode: npm run dev
PS D:\AI\claude-code-udemy-eazy-bytes\Section-3\Chapter-27\start\job-portal-ui> npm run dev



Access the Application in Browser
Once the server starts, you can access the app using URL: http://localhost:5173
Stop the Server
To stop the running application:
• Press Ctrl + C in the terminal
