Claude Code Native UI (VS Code) – Key Features

Automatically uses the open file as context when you ask a question.
    Example: "Explain the code in this file."
    Claude knows which file you're referring to.

Prioritizes the open file before searching the rest of the project.
    If needed, it automatically scans other files and folders for additional context.

Explain selected code only.
    Highlight specific lines of code and ask questions.
    Claude focuses only on the selected lines instead of the entire file.

Great for debugging and code reviews.
    Ask questions like:
    "Explain this code."
    "Find performance issues."
    "Suggest best practices."
    "Identify code smells or bugs."

Pinpointing code gives better answers.
    Selecting the exact lines helps Claude provide more accurate and relevant responses.

Option to disable the open file as context.
    You can tell Claude not to consider the currently open file when answering your prompt.

Drag and drop files into the chat.
    Hold Shift while dragging a file into the Claude UI.
    The file is automatically referenced using the @ notation.

Native UI provides features not available in the terminal.
    Automatic open-file context.
    Line selection as context.
    Disable file context with a click.
    Drag-and-drop file support.

Key takeaway
The Native UI in VS Code makes it much easier to ask context-aware questions by automatically understanding the file you're working on, the lines you've selected, and any files you drag into the chat. This results in faster, more precise explanations and code assistance compared to using the terminal alone.
